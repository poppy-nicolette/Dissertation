DOI: 10.1016/j.ipm.2023.103385
Title: 書誌情報 API の新しいトレンド: 比較分析 New trends in bibliometric APIs: A comparative analysis
Abstract: 科学の実践には、大規模で複雑な書誌データの分析が必要です。企業の Web サイトからエクスポートする従来のデータでは十分ではないため、API を使用してより大きなコーパスにアクセスします。したがって、この研究は、分類法を確立するだけでなく、さまざまな非営利および営利団体の 44 の書誌 API の比較分析を提供することも目的としています。記述分析による特性とメタデータの分析、可能な書誌分析、および 4 つの異なるデータ カテゴリ (一般、コンテンツ、検索、クエリ モード) にわたる API の相互運用性を分析します。この調査では、Clarivate Analytics と Elsevier が非常に汎用性の高い API を提供している一方、OpenCitations や OurResearch などの非営利組織がオープン サイエンスの理念を推進していることが判明しました。ほとんどの組織は非営利目的で API への無料アクセスを提供していますが、メタデータの取得に制限がある組織もあります。ただし、CrossRef、OpenCitations、または OpenAlex にはメタデータの取得に関する制限はありません。著者名を使用した共著者分析と引用を使用した書誌学的評価は、ほとんどの API によって提供されるデータを使用して実行できる分析の種類です。 DOI、PubMedID、および PMCID は、API でメタデータを拡張するための最も汎用性の高い識別子です。 Semantic Sc​​holar、Dimensions、ORCID、および Embase は、最も拡張性の高い API です。得られた結果を考慮すると、書誌学的分析を実行するために必要なすべての情報を収集する単一の API はありません。できるだけ多くの情報をカバーし、レポートと分析を充実させるには、2 つ以上の API を組み合わせることが最も適切なオプションである可能性があります。この研究は、研究実践における API の理解と使用の促進に貢献します。 The science of science practice requires the analysis of large and complex bibliometric data. Traditional data exporting from companies’ websites is not sufficient, so APIs are used to access a larger corpus. Therefore, this study aims not only to establish a taxonomy but also to offer a comparative analysis of 44 bibliographic APIs from various non-profit and commercial organizations, analyzing their characteristics and metadata with descriptive analysis, their possible bibliometric analyses, and the interoperability of the APIs across four different data categories: general, content, search, and query modes. The study found that Clarivate Analytics and Elsevier offer highly versatile APIs, while non-profit organizations, such as OpenCitations and OurResearch promote the Open Science philosophy. Most organizations offer free access to APIs for non-commercial purposes, but some have limitations on metadata retrieval. However, CrossRef, OpenCitations, or OpenAlex have no restrictions on the metadata retrieval. Co-author analysis using author names and bibliometric evaluation using citations are the types of analyses that can be done with the data provided by most APIs. DOI, PubMedID, and PMCID are the most versatile identifiers for extending metadata in the APIs. Semantic Scholar, Dimensions, ORCID, and Embase are the APIs that offer the most extensibility. Considering the obtained results, there is no single API that gathers all the information needed to perform any bibliometric analysis. Combining two or more APIs may be the most appropriate option to cover as much information as possible and enrich reports and analyses. This study contributes to advancing the understanding and use of APIs in research practice.
