Title: S3LLM：ソース、メタデータ、ドキュメントを使用したLLMSを使用した大規模な科学ソフトウェアの理解 S3LLM: Large-Scale Scientific Software Understanding with LLMs Using Source, Metadata, and Document
Abstract: 大規模な科学ソフトウェアの理解は、その多様なコードベース、広範なコード長、およびターゲットコンピューティングアーキテクチャのために、重要な課題です。生成AI、特に大規模な言語モデル（LLM）の出現は、このような複雑な科学コードを理解するための新しい経路を提供します。このペーパーでは、ソースコード、コードメタデータ、およびユーザーフレンドリーなインターフェイスを通じてインタラクティブで会話的な方法でテキスト技術レポートと併せて要約された情報の調査を可能にするために設計されたLLMベースのフレームワークであるS3LLMを紹介します。 S3LLMは、オープンソースLLAMA-2モデルを活用して、自然言語クエリのドメイン固有言語（DSL）クエリへの自動変換を通じてコード分析を強化します。さらに、S3LLMは、DOT、SQL、カスタマイズされた形式を含む多様なメタデータタイプを処理するために装備されています。さらに、S3LLMには、検索された生成（RAG）およびLangchain Technologiesが組み込まれ、広範なドキュメントを直接照会します。 S3LLMは、大規模な科学コンピューティングソフトウェアを迅速に理解するためにローカルに展開されたオープンソースLLMを使用する可能性を実証し、広範なコーディングの専門知識の必要性を排除し、それによりプロセスをより効率的かつ効果的にします。 S3llmはhttps://github.com/responsibleailab/s3llmで入手できます。 The understanding of large-scale scientific software is a significant challenge due to its diverse codebase, extensive code length, and target computing architectures. The emergence of generative AI, specifically large language models (LLMs), provides novel pathways for understanding such complex scientific codes. This paper presents S3LLM, an LLM-based framework designed to enable the examination of source code, code metadata, and summarized information in conjunction with textual technical reports in an interactive, conversational manner through a user-friendly interface. S3LLM leverages open-source LLaMA-2 models to enhance code analysis through the automatic transformation of natural language queries into domain-specific language (DSL) queries. In addition, S3LLM is equipped to handle diverse metadata types, including DOT, SQL, and customized formats. Furthermore, S3LLM incorporates retrieval-augmented generation (RAG) and LangChain technologies to directly query extensive documents. S3LLM demonstrates the potential of using locally deployed open-source LLMs for the rapid understanding of large-scale scientific computing software, eliminating the need for extensive coding expertise and thereby making the process more efficient and effective. S3LLM is available at https://github.com/ResponsibleAILab/s3llm.
