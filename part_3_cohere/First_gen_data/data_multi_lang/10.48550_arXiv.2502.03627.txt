Title: バベルのせせらぎの並べ替え：OpenAlexデータベースでの言語検出アルゴリズムのパフォーマンスの評価 Sorting the Babble in Babel: Assessing the Performance of Language Detection Algorithms on the OpenAlex Database
Abstract: このプロジェクトの目的は、さまざまな言語分類手順、さまざまなPython言語検出アルゴリズムと、オープンアレックスデータベースからサンプリングされた手動で発音された記事から抽出されたメタデータベースのコーパスを組み合わせた手順を比較することを目的としています。各アルゴリズム、コーパス、言語の精度とリコールのパフォーマンスの分析に続いて、各アルゴリズムとコーパスタイプに記録された処理速度の分析に続いて、データベースレベルでの全体的な手順パフォーマンスは、各アルゴリズム、コーパス、および言語と言語全体のバリックなモデルの確率論的モデルの確率モデルを使用してシミュレートされました。結果は、手順のパフォーマンスが、実装された各測定値に与えられる重要性に強く依存していることを示しています。貪欲なコーパスでlangidアルゴリズムを使用すると、精度が優先されるコンテキストの場合、最良の結果が得られます。ただし、リコールが少なくとも精度よりもわずかに重要であると見なされるすべての場合、または処理時間に何らかの考慮事項が与えられるとすぐに、FastSpellアルゴリズムとタイトルを組み合わせた手順は、他のすべての選択肢を上回ります。真に多言語の大規模な参考文献データベースがないことを考えると、これらの結果が、言語間、書誌ベースの研究と分析のためのOpenalexデータベースの比類のない可能性を確認し、育成することが期待されています。 This project aims to compare various language classification procedures, procedures combining various Python language detection algorithms and metadata-based corpora extracted from manually-annotated articles sampled from the OpenAlex database. Following an analysis of precision and recall performance for each algorithm, corpus, and language as well as of processing speeds recorded for each algorithm and corpus type, overall procedure performance at the database level was simulated using probabilistic confusion matrices for each algorithm, corpus, and language as well as a probabilistic model of relative article language frequencies for the whole OpenAlex database. Results show that procedure performance strongly depends on the importance given to each of the measures implemented: for contexts where precision is preferred, using the LangID algorithm on the greedy corpus gives the best results; however, for all cases where recall is considered at least slightly more important than precision or as soon as processing times are given any kind of consideration, the procedure combining the FastSpell algorithm and the Titles corpus outperforms all other alternatives. Given the lack of truly multilingual, large-scale bibliographic databases, it is hoped that these results help confirm and foster the unparalleled potential of the OpenAlex database for cross-linguistic, bibliometric-based research and analysis.
