Title: KNIMEZoBot: 検索拡張生成を使用した Zotero と KNIME OpenAI 統合による文献レビューの強化 KNIMEZoBot: Enhancing Literature Review with Zotero and KNIME OpenAI Integration using Retrieval-Augmented Generation
Abstract: 感染症の研究者は、自分たちの分野で急激に増加する発表結果に追いつくという課題に直面しています。知識を統合するために包括的な文献レビューを実行することは、手動のアプローチでは時間と労力がかかります。人工知能の最近の進歩は有望なソリューションを提供しますが、その多くはコーディングの専門知識を必要とし、アクセシビリティが制限されています。 KNIMEZoBot は、Zotero、OpenAI、および KNIME ビジュアル プログラミング プラットフォームを革新的に統合し、コーディング経験のないユーザーの文献レビュー タスクを自動化します。 KNIME の直感的なグラフィカル インターフェイスを活用することで、研究者は Zotero ライブラリを検索するためのワークフローを作成し、OpenAI モデルを利用してコーディングせずに重要な情報を抽出できます。ユーザーは API キーを提供し、ローカルに保存されたワークフローのコピーで使いやすいインターフェイスを介して設定を行うだけです。 KNIMEZoBot は、チャットボット経由で自然言語で質問できるようにし、論文から関連する文章を取得して合成回答を生成します。このシステムは、Zotero の個人ライブラリ内の出版物の検索と分析を自動化することで、コーディングに不慣れな研究者による文献レビューを迅速化する大きな可能性を秘めています。 KNIMEZoBot は、慎重に設計された AI ツールがどのようにアクセシビリティを拡張し、多様な研究領域にわたる知識構築を加速できるかを実証します。 demic researchers face challenges keeping up with exponentially growing published findings in their field. Performing comprehensive literature reviews to synthesize knowledge is time-consuming and labor-intensive using manual approaches. Recent advances in artificial intelligence provide promising solutions, yet many require coding expertise, limiting accessibility. KNIMEZoBot represents an innovative integration of Zotero, OpenAI, and the KNIME visual programming platform to automate literature review tasks for users with no coding experience. By leveraging KNIME's intuitive graphical interface, researchers can create workflows to search their Zotero libraries and utilize OpenAI models to extract key information without coding. Users simply provide API keys and configure settings through a user-friendly interface in a locally stored copy of the workflow. KNIMEZoBot then allows asking natural language questions via a chatbot and retrieves relevant passages from papers to generate synthesized answers. This system has significant potential to expedite literature reviews for researchers unfamiliar with coding by automating retrieval and analysis of publications in personal Zotero libraries. KNIMEZoBot demonstrates how thoughtfully designed AI tools can expand accessibility and accelerate knowledge building across diverse research domains.
